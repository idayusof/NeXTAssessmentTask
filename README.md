# NeXTAssessmentTask
This is a command line application created for NeXT Assessment to find the location of International Space Station (ISS) at a certain date and time.
Extension B is implemented in this assessment by returning the information through an API call. 
This application is developed using C# language

How to run the program:
1. Clone the repository by copying the link from GitHub
2. Open Visual Studio 2019 and choose 'Clone a Repository'
3. Paste the link that has been copied from GitHub
4. Choose a local path to save the project
5. Once the repository has been cloned, click on 'View' at the top left and click on 'Solution Explorer' or simply press Ctrl+Alt+L
6. Click on 'NeXTAssessmentTask.sln' and then 'Program.cs' from Solution Explorer
7. Once the code has appeared, run the program by clicking the 'NeXTAssessmentTask' at the top middle with a green arrow besides it or simply press F5
8. The program should be running now.
